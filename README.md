Descrição: 
Notícias esportivas são conteúdos jornalísticos voltados à cobertura de tudo o que acontece no universo dos esportes. Esse tipo de notícia pode abranger tanto eventos locais quanto competições internacionais, envolvendo diversas modalidades, como futebol, basquete, vôlei, entre outras. Nosso projeto será dedicado a criação de uma plataforma digital dedicada a informar, analisar e entreter o público apaixonado por esportes. Ele reúne conteúdos atualizados em tempo real sobre diversas modalidades, atendendo tanto fãs casuais quanto entusiastas mais aprofundados. 

Tecnologia do FrontEnd:
Next.JS: O Next.js é um framework de desenvolvimento web baseado em React que permite criar aplicações modernas de forma mais rápida e eficiente. Ele oferece recursos como renderização no lado do servidor (SSR), geração de páginas estáticas (SSG) e roteamento automático, o que facilita a construção de sites performáticos e bem estruturados.
